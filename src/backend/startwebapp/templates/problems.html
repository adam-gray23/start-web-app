Problem 1: Hello World!

Write a program that outputs the message “Hello World” using the built-in write() function.
Target output:

Hello World

Problem 2: Your Name Here

Write a program that outputs your own name and age in the message: "My name is <name> and I am <age>". Make use of the variables "name" and "age" to store this information.
Target output:

My name is <name> and I am <age>

Try it yourself
Problem 3: Bank Balance

Write a program to mimic the way in which money enters and leaves your bank account. Define your initial variable, balance, to be 100. Then, using each of these variables, update the value of the variable balance. At the end output your final balance. Hint: Payments are subtracted to your balance while incomes are added. Here is a list of 5 transactions to update with:

  • rent is 50 (payment)
  • coffee is 3 (payment)
  • payCheck is 100 (income)
  • schoolSupplies is 25 (payment)
  • moneyTransfer is 30 (income)
Remember you can update a variable like follows:

x is x
x is x + 1

Target output:

152

Try it yourself
Problem 4: Even or Odd

Write a program to determine if a defined variable x, is even or is odd. Make use of an if/otherwise statement to solve this problem. Your output can be one of the following:
Target output:

x is even OR x is odd

Try it yourself
Problem 5: Temperature Converter

Write a program that converts temperature from Celsius to Fahrenheit. The steps to convert to Fahrenheit are as follows:

  • Take your original number and multiply by 1.8
  • Then add 32 to the result
Make use of a variables named "fahrenheit" and "celsius". Celsius should begin at 20. Your program should make use of the "mult" and "add" operators.
Target output:

68.0

Try it yourself
Problem 6: List Operations

Given we start with a list with the following value: [1,2,"x",1,2,3,"x",4,6,"x"]. As we can see, this list has gotten corrupted. We have the list repeating after the first 2 numbers, as well as a 6 instead of 5 at the end of the list. As well as these we have many corrupted x’s throughout. Make a program that removes all of these unwanted characters, and adds 5 correctly to the end of the list. Finally output the corrected list.
Hint: Make use of the remove and remove all function
Target output:

[1, 2, 3, 4, 5]

Try it yourself
Problem 7: Is List Index Even Or Odd

Given the following variable: list, [1,3,4,6,7,8,9], check each index and write “Even” if the number is even, and “Odd” if the number is odd. Your program should make use of a loop for each and an if/otherwise statement to solve this problem.
Target output:

Odd Odd Even Even Odd Even Odd

Try it yourself
Problem 8: Count Evens

Define variables "index" to be 0 and "target" to be 20. Use a loop to check "index" to see if it is even. If it is increment another variable "count" by 1. At the end of the loop increment "index" by 1. Repeat this process until "index" is equal to 20. I.E. do not run the loop if "index" is equal to 20.
Target output:

10

Try it yourself
Problem 9: Reuse Add 2

Write a program which uses a function named "addTwo" to add two numbers together. Your program should contain the code:
addTwo(2, 2)
addTwo(5, 3)
Target output:

4 8

Try it yourself
Problem 10: Recursion Challenge - Fibonacci Sequence

In the Fibonacci sequence, each number is the sum of the two preceding ones, starting from 0 and 1. The task is to write a recursive function called fibonacci that takes an integer n as an argument and returns the nth Fibonacci number. Your program should calculate the 4th and 10th fibonacci numbers.
E.g.
fibonacci(4)
fibonacci(10)
More information about the fibonacci sequence can be found here.
Target output:

3 55

Try it yourself
